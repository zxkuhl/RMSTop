## What is RMSTop?
RMSTop is a standalone desktop client that comes with [RMS](https://github.com/zxkuhl/RMS) built in, giving you a snappier experience with all of RMS's plugins, themes, and customization out of the box — no manual setup required.

<p align="center">
  <img src="https://raw.githubusercontent.com/zxkuhl/RMSTop/main/icons/ui.png" alt="RMSTop UI" width="800" />
</p>

## Features
- Full [RMS](https://github.com/zxkuhl/RMS) integration out of the box
- Auto-updates for both RMSTop and RMS
- Custom splash screen and theming
- Windows transparency support
- Tray icon with minimize to tray
- Custom title bar option
- arRPC rich presence support

## RMS Files Setup

1. Download the latest RMS files from the link below:
   **[RMSTop RMS Files - Latest Release](https://github.com/zxkuhl/RMSTop/releases/tag/1.22.0)**

2. Create the following folder path if it doesn't exist:

```
%AppData%\rmstop\sessionData\rmsFiles\
```

3. Place all downloaded files into that folder. It should look like this:

```
rmsFiles/
├── package.json
├── rmsDesktopMain.js
├── rmsDesktopPreload.js
├── rmsDesktopRenderer.css
└── rmsDesktopRenderer.js
```

4. Restart RMSTop.

> Once set up, RMSTop will automatically keep these files up to date.
